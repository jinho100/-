# 학술 논문 연령별 눈높이 요약 프롬프트 빌더 (Paper Simplifier Prompt Builder)

어려운 학술 논문이나 전문적인 연구 자료를 사용자가 지정한 연령대의 이해 수준에 맞춰 쉽게 풀어쓰도록 유도하는 **LLM(대형 언어 모델) 전용 프롬프트 생성기**입니다.

이 도구는 복잡한 프롬프트 엔지니어링 과정 없이, 논문 텍스트와 대상 연령대만 입력하면 ChatGPT, Claude 등의 언어 모델에 최적화된 구조화 프롬프트를 자동으로 조립해 줍니다.

---

## 주요 기능 (Features)

- **연령별 맞춤 가이드라인 제공**: 초등 저학년(7~10세)부터 일반 성인(비전공자)까지 총 5가지 단계의 인지적 발달 수준에 부합하는 비유와 설명 구조를 내장하고 있습니다.
- **HTML 기반 구조화 태그 적용**: LLM이 지시사항을 정확하고 계층적으로 파악할 수 있도록 XML/HTML 스타일의 태그를 활용한 프롬프트를 생성합니다.
- **간편한 실행 환경**: 별도의 패키지 설치나 개발 환경 설정 없이, 하나의 독립된 HTML 파일만 웹 브라우저로 열어 즉시 사용할 수 있습니다.
